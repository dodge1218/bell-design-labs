# UX Flow Summary

1. **Homepage → Pricing** — Hero clarifies the build sprint, pain points build urgency, and comparison table nudges visitors to Pricing for proof of value.
2. **Pricing → Calculator** — Plans show transparent costs, add-ons stay collapsed until expanded, and users jump into the calculator to tailor totals.
3. **Calculator → Contact / Start Project** — Live totals feed into the CTA, encouraging either a quick contact or full intake submission.
4. **Work → Contact** — Case study and stats add social proof before presenting the contact CTA and testimonials.
5. **Contact & Start Project** — Forms collect quick questions or detailed scope with RHF/Zod validation and clear "what happens next" messaging so leads know when to expect a reply.
